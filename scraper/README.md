# ZevenBots Scraper Engine

This folder is dedicated entirely to lead generation and web scraping. 

Do not mix frontend React code into this folder. Keep all Python, Playwright, Node.js, and Firecrawl scraping scripts isolated here.

## Contents
- `prospects.md`: The initial list of leads and targets generated via Firecrawl.
- Future scripts (e.g. `scrape_maps.js` or `apify_webhook.py`) should be saved here.
